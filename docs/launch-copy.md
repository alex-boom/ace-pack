# ACE Launch Copy

Use these snippets when publishing a release, writing a launch post, or asking
for feedback from AI tooling communities. Keep the claim simple: ACE is local
memory and guardrails for AI coding agents.

## One-Line Positioning

Zero-dependency local AgentOps for AI coding agents: durable memory, guardrails,
and quality gates for vibe coding in real repositories.

## Short Description

ACE Pack turns a repository into a better working environment for AI coding
agents. It installs plain Markdown memory, repository-aware risk rules, focused
context generation, and optional PR/CI gates without adding runtime
dependencies, SaaS accounts, model lock-in, or hidden AI calls.
